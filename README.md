# 세종 · 세종특별자치시

> 세종 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-sejong-sejong.github.io/**

세종특별자치시는 대한민국의 유일한 특별자치시이자 실질적인 행정수도 기능을 담당하는 광역자치단체다. 2012년 7월 1일 충청남도 연기군 전역과 공주시·충청북도 청원군 일부를 통합해 대한민국 17번째 광역자치단체로 출범했으며, 국토 균형발전과 수도권 과밀 해소를 위해 조성된 계획도시 '행정중심복합도시(행복도시)'를 품고 있다. 어진동 정부세종청사에는 국무조정실을 비롯한 다수 중앙행정기관과 국책연구단지가 자리해, 국무회의·부처 업무가 세종에서 이뤄지는 사실상 제2의 수도 역할을 한다. 면적은 약 465.96㎢로 서울의 약 70% 규모지만, 남쪽 신도시권과 북쪽 조치원·읍면 지역이 뚜렷이 구분되는 이원적 도시 구조가 특징이다. 금강과 미호강이 시가지를 관통하고 세종호수공원·국립세종수목원·금강보행교 이응다리 등 대규모 친수·녹지 인프라가 밀집해 '정원도시'로도 불린다. 청년·공무원 유입으로 전국 최고 수준의 인구 증가율과 낮은 평균연령을 기록하며 빠르게 성장하는 젊은 도시다.
